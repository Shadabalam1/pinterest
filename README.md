# 📌 Pinterest Clone (Backend)

A backend web application inspired by Pinterest, built using Node.js, Express, and MongoDB. It supports user authentication, session management, and file uploads.

---

## 🚀 Features

* 🔐 User Authentication (Login / Register)
* 🔑 Session Management using Express Session
* 📸 File Uploads using Multer
* 🗄️ MongoDB Database with Mongoose
* 🎨 Server-side rendering using EJS
* ⚡ Clean Express.js architecture

---

## 🛠️ Tech Stack

* Node.js
* Express.js
* MongoDB (Mongoose)
* EJS
* Passport.js (Authentication)
* Multer (File Upload)

---

## ▶️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/pinterest.git
```

### 2. Navigate to project folder

```bash
cd pinterest
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the server

```bash
npm start
```
